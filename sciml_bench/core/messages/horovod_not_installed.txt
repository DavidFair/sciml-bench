
********************* Horovod Installation *********************
Horovod is required by one or some of the selected benchmarks
for distributed learning, but it has not been installed.

Based on the selected benchmarks, the suggested command line
to install Horovod is

$ HOROVOD_CMD_STR

You should choose "NO" if you want to install Horovod with more
environment variables (e.g., HOROVOD_GPU_OPERATIONS,
HOROVOD_WITH_MPI, etc.). See here for a guidance of all
Horovod environment variables:
https://horovod.readthedocs.io/en/stable/install_include.html
